# 🌦️ AI Weather-Smart Assistant (n8n)

An automated workflow built with **n8n** that fetches live weather data and uses AI to provide personalized clothing advice.

## 🚀 Features
- **Daily Updates:** Scheduled to run every morning.
- **AI-Powered:** Uses logic to suggest outfits (e.g., "Wear cotton" if > 30°C).
- **Direct Alerts:** Sends a clean email via Gmail.

## 🛠️ Tools Used
- **n8n** (Automation Platform)
- **OpenWeatherMap API** (Weather Data)
- **Gmail Node** (Communication)
- **AI Transform Node** (Personalized Advice)

## 📦 How to Use
1. Download the `Automated Personalized Weather Emailn.json` file from this repo.
2. Import it into your n8n instance.
3. Add your OpenWeatherMap API key and Gmail credentials.
4. Hit **Execute**!
